1. Pengertian prinsi single responsibility dan open closed serta keuntungannya
- Prinsip single responsibilty merupakan sebuah prinsip dimana suatu class
hanya memiliki satu fungsi atau tugas yang dijalankan. 
Keuntungan dari single responsibility yakni source code lebih rapi dan membuat 
code sistem lebih efisien dan responsive

- Prinsip open closed merupakan sebuah prinsip dimana kita dapat menutup source code utama
namun dapat diubah dengan menambah entitas  tanpa harus merubah sumber source code utama.
Keuntungan dari open closed yakni dapat meminimalisir kerusakan sumber source code utama
akibat dari pengubahan code
